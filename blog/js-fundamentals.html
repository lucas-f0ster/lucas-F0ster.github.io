What's up with that HTML and CSS stuff?

Most websites are made of HTML and CSS files, at their most basic.
They look very similar, and work very closely with each other but the difference is basically this:

HTML is a sort of framework, a list. It's a document containing all the differnt pieces of text, images, boxes, buttons and the likes that make up the webpage. It's no more than a text document, really, a list of different elements and a rough idea of which ones are inside the others.

That's why we we create some styles with our CSS document.

On its own, a CSS file is even less exciting than an HTML document, at least we HTML we get an ugly list of parts while with CSS, you wouldn't see anything at all!

CSS files exist to tell the HTML Elements how they should look, where they should sit on the page and where they should sit compared to each other

In short, HTML describes WHAT is in your page and CSS describes HOW that stuff should be shown!
